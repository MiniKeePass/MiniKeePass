Ajouter des fichiers via Dropbox

Les fichiers KeePass stockés dans Dropbox peuvent être ajoutés à MiniKeePass en utilisant l'application Dropbox.

Etapes:

  • Ouvrez Dropbox sur votre périphérique
  • Naviguez jusqu'à votre fichier KeePass
  • Tapez le bouton Action de la barre d'outils
  • Sélectionnez MiniKeePass dans la liste des applications

Téléverser des fichiers dans Dropbox

MiniKeePass ne synchronise pas automatiquement les fichiers avec Dropbox. Suivez les étapes suivantes pour exporter vos fichiers KeePass vers l'application Dropbox.

Etapes:

  • Ouvrez MiniKeePass sur votre périphérique
  • Ouvrez le fichier base de données à exporter
  • Tapez le bouton Action de la barre d'outils
  • Sélectionnez Dropbox dans la liste des applications
