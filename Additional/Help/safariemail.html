Opening files from Safari

MiniKeePass can open KeePass files from Safari, Email, or any other app that presents the user with the option of opening KeePass files.

Steps:

  • Navigate to the page with the KeePass file in Safari
  • Tap on the link to your KeePass file
  • Select Open in "MiniKeePass"

Opening Email Attachments

Steps:

  • Open an email with a KeePass file attached
  • Tap on the KeePass attachment
  • Select Open in "MiniKeePass"
