Creating a new password database

MiniKeePass can create new KeePass files in either the orginal 1.x format, or the newer 2.x format

Steps:

  • Tap the "+" button on the main screen
  • Choose a name for your new file
  • Enter and confirm the master password
  • Choose Version 1.x or Version 2.x
  • Tap "Done" in the upper right corner
