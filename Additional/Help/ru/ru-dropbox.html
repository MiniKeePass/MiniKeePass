Добавление файлов из Dropbox

Файлы KeePass, находящиеся в Dropbox, могут быть добавлены в MiniKeePass, используя приложение Dropbox.

Шаги:

  • Откройте приложение Dropbox на вашем устройстве
  • Найдите ваш файл KeePass
  • Нажмите кнопку Действия в панели инструментов
  • Выберите MiniKeePass из появившегося списка

Загрузка файлов в Dropbox

MiniKeePass не может автоматически синхронизировать файлы с Dropbox. Следуйте шагам для экспорта ваших файлов KeePass в приложение Dropbox.

Шаги:

  • Откройте MiniKeePass на вашем устройстве
  • Откройте экспортируемый файл
  • Нажмите кнопку Действия в панели инструментов
  • Выберите Dropbox из появившегося списка
