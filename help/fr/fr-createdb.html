Créer une nouvelle base de données

MiniKeePass peut créer de nouvelles bases de données KeePass dans le format original 1.x, ou dans le nouveau format 2.x

Etapes:

  • Tapez le boutton "+" sur l'écran principal
  • Nommer le fichier base de données
  • Saisir et confirmez le mot de passe maître
  • Choisir la Version 1.x ou Version 2.x
  • Tapez "Terminé" dans le coin supérieur droit
