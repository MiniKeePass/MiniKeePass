NEW! -- Link directly to Dropbox files within the App

Enable Dropbox Integration in the settings. The dropbox app or web browser will open and ask you to login and allow MiniKeePass to access your files.

MiniKeePass can only see files that are put in the Apps/MiniKeePass directory.

Key files are not yet supported.

Adding files from Dropbox (Old Way)

KeePass files located on Dropbox can be added to MiniKeePass using the Dropbox app.

Steps:

  • Open the Dropbox app on your device
  • Navigate to your KeePass file
  • Tap the Action button in the toolbar
  • Select MiniKeePass from the list that appears

Uploading files to Dropbox

MiniKeePass will not automatically sync files with Dropbox unless you use Dropbox integration in the settings. Follow these steps to export your KeePass files to the Dropbox app.

Steps:

  • Open MiniKeePass on your device
  • Open the file to be exported
  • Tap the Action button in the toolbar
  • Select Dropbox from the list that appears
