Importing from Dropbox

KeePass files located in Dropbox can be added to MiniKeePass using the Dropbox app. MiniKeePass will not automatically fetch the database file when it is updated on Dropbox

  1. Open Dropbox.
  2. Navigate to your KeePass database file and open it.
  3. Tap the ... icon in the top right hand corner.
  4. Tap "Export"
  5. Select "Open In..."
  6. Find "Copy to MiniKeePass" and tap it.
  7. Your KeePass database will open in MiniKeePass and now be listed under "Databases"

Exporting to Dropbox

MiniKeePass will not automatically sync files with Dropbox. Follow these steps to export your KeePass files to the Dropbox app.

  1. Open MiniKeePass.
  2. Open the database you want to export.
  3. Tap the action button on the bottom toolbar.
  4. Select "Import with Dropbox"
  5. Choose a location on the Dropbox app and tap save.
