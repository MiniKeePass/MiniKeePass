Importe arquivos do Dropbox

Arquivos do KeePass localizados no Dropbox podem ser adicionados ao MiniKeePass através do aplicativo Dropbox.

Passos:

  • Abra o aplicativo Dropbox no dispositivo;
  • Navegue até o arquivo do KeePass;
  • Toque no botão de compartilhamento;
  • Selecione MiniKeePass na lista exibida.

Exporte arquivos para o Dropbox

O MiniKeePass não sincroniza arquivos com o Dropbox automaticamente. Siga estes passos para exportar arquivos do MiniKeePass para o aplicativo Dropbox.

Passos:

  • Abra o MiniKeePass no dispositivo;
  • Abra o arquivo que deseja exportar;
  • Toque no botão de compartilhamento;
  • Selecione Dropbox na lista exibida.
