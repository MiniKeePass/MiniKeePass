Use archivos llave

MiniKeePass puede abrir archivos KeePass 1.x/2.x que usen un archivo llave en lugar o junto a una contraseña.

Pasos:

  • Cargue su base de datos y su archivo llave en MiniKeePass usando iTunes, Dropbox, etc...
  • Abra el archivo KeePass en MiniKeePass
  • Cuando se le solicite una contraseña, podrá introducirla y/o seleccionar un archivo llave

Nota: MiniKeePass seleccionará automáticamente su archivo llave si este tiene el mismo nombre que su KeePass (pero con extension .key).
